# Kisan Alert Brief Description

Kisan Alert is a voice-first crop guidance prototype designed for farmers in tier 2 and tier 3 India, where shared phones, mixed literacy, and local-language support are critical. Instead of behaving like a generic chatbot, it reduces the experience to four surfaces: `Home`, `Ask`, `Alerts`, and `Account`.

The product flow focuses on short, farmer-safe outputs. A farmer can ask for help by voice, image, or text, and the system is intended to return one clear summary, a few next actions, and explicit escalation when confidence is low. The current repository contains an Android prototype plus a farmer-facing web companion that demonstrates this interaction model.

The long-term production direction is a service-backed advisory system with server-side authentication, structured response contracts, retrieval over approved agronomy content, and audit logging. The current submission represents a practical, narrowly scoped prototype built around deployability, trust, and usability rather than feature breadth.
